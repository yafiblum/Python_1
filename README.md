# Number Guessing Game

This is a simple Python game where the player tries to guess a randomly generated number between 1 and 20 within 6 attempts.

## How to Play
- Run the script `main.py`.
- The game will prompt you to guess a number between 1 and 20.
- You have 6 attempts to guess the correct number.
- After each guess, you will receive feedback:
  - "high" if your guess is too low
  - "low" if your guess is too high
  - "correct" if you guessed the number
- If you guess correctly, you win! Otherwise, the correct number will be revealed after 6 attempts.

## Requirements
- Python 3.x

## Running the Game
```bash
python main.py
```

Enjoy the game!
